#### 项目技术栈
前端：JavaScript、UI组件Element-Plus、框架Vue、CSS 3、HTML 5、
后端：框架Spring Boot、数据持久化MyBatis Plus、认证授权机制Jwt、缓存技术Redis、数据库MySQL、数据库分片Sharding-JDBC、消息队列RabbitMQ
其他：git
